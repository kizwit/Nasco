## Requirements

- Python 3.9 or higher
- Scapy 2.4.5 or higher
- Nslookup {optional}
- Linux or Windows

## Usage

To use the project, follow these steps:

- Open the terminal and run the following command:
- `cd ddnasco`
- `pip install -r requirements.txt`
- `python3 ddnasco.py`

  
- Thats it! The test will start automatically.
